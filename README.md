# Visualizador de Árvore B+ 

## Descrição

Projeto desenvolvido para a disciplina de Estrutura de Dados com o objetivo de demonstrar, de forma visual e interativa, o funcionamento de uma Árvore B+ , utilizando apenas HTML, CSS e JavaScript .

Além da implementação, o repositório também inclui um slide em PDF com a explicação da Árvore B+, utilizada na apresentação do projeto.
