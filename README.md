# Visualizador de Árvore B+ 

## Descrição

Projeto desenvolvido para a disciplina de **Estrutura de Dados** com o objetivo de demonstrar, de forma visual e interativa, o funcionamento de uma **Árvore B+**, utilizando apenas **HTML, CSS e JavaScript**.

Além da implementação, o repositório também inclui um **slide em PDF** com a fundamentação teórica da Árvore B+, utilizado na apresentação do projeto.

## Estrutura do Repositório

```text
/
├── Implementação/
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── Slide/
│   └── Arvores-B+.pdf
│
└── README.md
```

## Funcionalidades

* Criar árvore
* Inserir chaves
* Buscar chaves
* Remover chaves
* Destruir a árvore
* Demonstração automática das operações

## Slide

O arquivo **Arvores-B+.pdf** apresenta os principais conceitos da Árvore B+, incluindo:

* Introdução
* Estrutura do nó
* Inserção, busca e remoção
* Complexidade das operações
* Vantagens e desvantagens
* Comparação com outras árvores

## Tecnologias

* HTML5
* CSS3
* JavaScript

## Como Executar

Abra o arquivo **`Implementação/index.html`** em um navegador para utilizar o visualizador. Para consultar a parte teórica, acesse o arquivo **`Slide/Arvores-B+.pdf`**.
